# Sales Analysis Dashboard - Power BI

This project features a comprehensive Power BI dashboard for sales analysis, offering insights into product performance, inventory, and future trends.

## Key Features

- **Sales Overview:** Visualize sales by category, identify top products, and monitor performance.
- **Inventory Management:** Receive low-stock alerts to maintain optimal inventory levels.
- **Forecasting:** Utilize predictive models to forecast future sales trends.
- **Detailed Analysis:** Use decomposition trees and interactive filters for drill-down analysis.
